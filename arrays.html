Types of arrays


1.PUSH: The Array. prototype.push() Method is used to add one or more values to the end of the array.
2.POP: The pop() method removes (pops) the last element of an array.
3.UNSHIFT: The unshift() method adds new elements to the beginning of an array.
4.SHIFT: The shift() method removes the first element from an array and returns that element.
